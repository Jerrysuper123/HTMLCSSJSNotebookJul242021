the form properties, below 4 form main properties for form

  1. action - which url to submit to
  2. method - Get or Post metho
  3. name - rarely used, usually we use ID attribute to select
  4. elements - a collection of elements within form; how to access
    1. you can access form by array index or name attribute
    2. e.g. document.forms[1] - access first form
    3. document.forms.login - access the form by its name attribute
    4. you can access form's elements (inner elements) by array index or name attribute
    5. document.form[1].elements[0]
    6. document.forms[1].elements.password

form method

  1. submit method
  2. reset method to reset the input to inital value

form control uses a combination of properties, methods and events

property

  1. value - value of text input
  2. type - it could be text, password, submit button, radio, or check box
  3. name - name attribute that you set
  4. defaultValue - placeholder when page initally loaded
  5. diabled - true or false to disble the form
  6. checked - true or false to see if checkbox or radio button have been clicked
  7. selected - true or false to see if a select box has been selected

method

  1. focus() - focus on an input
  2. blur() - leave an input
  3. select() - select or highlight content of an element
  4. click() - trigger a click even

event

  1. blur - leave a field
  2. focus - enter a field
  3. change - when input value changes
  4. inout - when input of input or textarea changes
  5. keyDown, KeyUp, keyPress
