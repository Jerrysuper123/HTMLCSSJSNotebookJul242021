intro - 6 types of events/changes that could trigger script

  1. Ui event - load/unload/scroll/resize
  2. keyboard - keydwon, keyUp
  3. mouse - click, dclick, mousedown, mouseup
  4. focus - focus, blur (when in the form)
  5. form events - input changes, change, submit

triggering process

  1. select node
  2. what happened to the node - choose events (binding - binding an event to dom node)
  3. call code based on event

3 ways of binding

  • html event handler (do not use)
  • DOM event handler - can only trigger one function el.onEvent = functionName
  • event listener - can trigger multiple functions el.addEventLisentner(event, function, booean (flow))
  • Event flow - false (event bubbling specifc to less specific true (event capturing) - event generic to specific
  • event can be passed into function as a parameter
  • event delegation - find parent node for event, the delegate to its children
  • preventDefault - when you submit form, it will take you to another page, this function stop it
  • stopPropagation - stop flow either event bubbling/event capturing
