transition is animation, first start with 2 arguments first one specific what function to transition (transform in our case), and how many second to achieve the transition

always put the transition into the element that you want to animate

Timing function: the third argument is how you want the element to move

  • linear - move linearly
  • ease-in-out, start slow (ease speed), pick up speed in mode, out of speed

the fourth argument is the delay - if you put 3 second, it will delay 3 seconds before moving

  • arg 1 - what transform
  • arg 2 - move many seconds
  • arg 3 - how to move - linear, ease in out,
  • arg 4 - delay
